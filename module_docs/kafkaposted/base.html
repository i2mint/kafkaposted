kafkaposted.base

Base module for Apache Kafka message broker.

This module provides an implementation of the MsgBrokerBase interface for Apache Kafka. The broker is implemented using the kafka-python package. The broker is designed to be used in a multi-threaded environment, and it uses a thread pool to consume messages from the Kafka topics.
